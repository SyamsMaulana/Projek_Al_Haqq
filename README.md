# Jurnal Ilmiah Terbuka: Projek Al-Haqq v1.6
### Integrasi Ilmu Psikiatri dan Teknologi Informasi untuk Studi Bipolar & Schizophrenia

---

## 1. Pengantar & Visi
Projek Al-Haqq adalah sebuah repositori jurnal ilmiah terbuka yang didedikasikan untuk menjembatani jurang pemisah antara metodologi klinis konvensional dengan pemanfaatan teknologi informasi modern. Jurnal ini berfokus pada pemanfaatan *Data Science*, *Natural Language Processing* (NLP), dan arsitektur informasi untuk mendukung hasil riset global yang berbasis pada studi langsung (*first-hand experience*).

Dengan menggunakan pendekatan **Inverted Pyramid Framework v1.6**, kami menempatkan data otentik dari akar rumput sebagai fondasi utama guna mengedukasi dan menyuplai pengetahuan bagi para peneliti, akademisi, dan praktisi medis di tingkat atas.

## 2. Ruang Lingkup Riset
* **Linguistik Komputasional:** Analisis pola perubahan gaya bahasa pada fase kognitif tertentu.
* **Privasi & Anonimitas Data:** Pengembangan skrip enkripsi dan pembersihan data sensitif (Anonymizer).
* **Teknologi Asistif:** Pemanfaatan tools open-source (Python/Termux) untuk pemetaan mandiri dinamika mental.

## 3. Kode Etik & Integritas (Al-Haqq)
Seluruh data, artikel, dan skrip yang dipublikasikan di sini wajib mematuhi standar privasi internasional dan ditujukan murni untuk kemaslahatan ilmu pengetahuan serta kemanusiaan.
